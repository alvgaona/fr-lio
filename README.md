> [!NOTE]
> This project builds on [FAST-LIO2](https://github.com/hku-mars/FAST_LIO) by the HKU MARS Lab
> (Wei Xu, Yixi Cai, Dongjiao He, Jiarong Lin, Fu Zhang). Credit for the foundational LiDAR-inertial
> odometry framework belongs to them; this fork extends their work with flight-readiness fixes,
> FEJ-IESKF observability consistency, and scan-to-scan drift covariance estimation.
>
> If you use this code, please also cite the original FAST-LIO papers:
>
> - W. Xu and F. Zhang, "FAST-LIO: A Fast, Robust LiDAR-Inertial Odometry Package by
>   Tightly-Coupled Iterated Kalman Filter," *IEEE Robotics and Automation Letters*, 2021.
> - W. Xu, Y. Cai, D. He, J. Lin, F. Zhang, "FAST-LIO2: Fast Direct LiDAR-Inertial Odometry,"
>   *IEEE Transactions on Robotics*, 2022.
